	Структуры в Си
Структуры — тип данных, задаваемый пользователем, который может содержать в себе несколько переменных разных типов.

	Объявление структуры типа Struct_Name, состоящей из целого числа Number и символа Symbol:
struct Struct_Name
{
	int Number;
	char Symbol;
};

	Объявление структуры a типа Struct_Name и инициализация её элементов:
struct Struct_Name a = {64, t};
Теперь структура a состоит из числа 64 и символа t

	К элементам этой структуры теперь можно обращаться как a.Number и a.Symbol.

	Удобство структур заключается в том, что можно объединять переменные, которые используются для одинаковых целей, например можно объединить координаты точки на плоскости или в пространстве в структуру.
