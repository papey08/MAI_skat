Сложность по времени (N — кол-во элементов исходного массива, M — наибольшая длина возрастающей подпоследовательности):
1) Считываем n элементов в массив, сложность O(N)
2) С помощью алгоритма ищем подпоследовательность наибольшей длины, сложность O(N)
3) Выводим найденную подпоследовательность, сложность O(M)
4) Общая сложность O(2*N + M)

Сложность по памяти:
Т.к. изначально задан массив, состоящий из 1001 элемента, сложность по памяти не зависит от N или M и равна O(1001).
